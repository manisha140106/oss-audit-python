## Project Overview

This project is about looking at Python, which's an open source software. We want to know where Python came from what it is about how it is licensed and how people use it on Linux. The project also shows that we really understand how to use Python by writing scripts for the shell.

##Chosen Software

Python. This is a programming language that's open source and easy to understand. It is known for being simple and easy to read.

##Scripts Included

1. Script1.sh. This script tells us about our system. It shows us information like who the user's what version of the kernel we have how long the system has been running what the date is and some details about Python.

2. Script2.sh. This script checks if Python is installed on our system. If it is the script tells us what version of Python we have and gives us some information about it.

3. Script3.sh. This script looks at the folders on our system. It shows us what permissions these folders have and how space they are taking up on the disk.

4. Script4.sh. This script reads a log file. Counts how many times a certain word appears in it like the word "error". Then it shows us the results.

5. Script5.sh. This script generates a statement about open source software just for us. It asks us for some information. Then creates a personalized open source philosophy statement.
