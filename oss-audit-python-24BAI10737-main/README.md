## Project Overview
This project is an Open Source Software Audit of Python. It explores the origin, philosophy, licensing, and real-world usage of Python in Linux environments. The project also demonstrates practical understanding through shell scripting.

## Chosen Software
Python – An open-source, high-level programming language known for simplicity and readability.

## Scripts Included

### 1. script1.sh – System Identity Report
Displays system information such as user, kernel version, uptime, date, and Python details.

### 2. script2.sh – Python Package Inspector
Checks whether Python is installed and displays its version along with basic information.

### 3. script3.sh – Disk and Permission Auditor
Analyzes important system directories and shows their permissions and disk usage.

### 4. script4.sh – Log File Analyzer
Reads a log file, counts occurrences of a keyword (like "error"), and displays results.

### 5. script5.sh – Open Source Manifesto Generator
Generates a personalized open-source philosophy statement based on user input.
